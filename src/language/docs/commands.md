RND
---
***INTEGER***
* `RND Number%`

Get a random number. If Number% is `0`, last random number will be returned.

CHR$
---
***STRING***
* `CHR$(Number%)`

Produce a STRING with a character corresponding to the `Number%` ASCII code value.

PRINT
---
* `PRINT "Hello World"`
* `PRINT "DUPA"`

Outputs the provided values onto the screen
